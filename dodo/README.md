## Run the app

### Android studio

![](demo/android.png)

### Visual Studio code

1. Select a device
![](demo/code.png)

2. Invoke Run > Start Debugging or press F5


### Terminal

1. Check devices

```bash
flutter devices
```

2. Run the app

```bash
flutter run
```
